# Manta-admin

This is the standalone Manta-admin dashboard project.

## Development

Open `index.html` or `dashboard.html` in your browser.

## Deployment

### Manual Deployment (Vercel/Netlify)

1. **Drag and Drop**: 
   - Drag the `Manta_web_userMgt` folder (or the provided zip contents) directly into the Vercel or Netlify dashboard.
   - Project Name: `Manta-admin`

2. **CLI Deployment**:
   - Run `npx vercel deploy --prod` inside this directory.
